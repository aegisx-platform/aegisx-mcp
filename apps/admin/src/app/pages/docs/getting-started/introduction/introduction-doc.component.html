Welcome to AegisX

AegisX is a modern design system that combines the best of Angular Material 3 with custom components inspired by Tremor. It provides a complete set of design tokens, components, and patterns to help you build enterprise-grade applications quickly and consistently.

What is AegisX Design System?

AegisX is not just a component libraryâ€”it's a comprehensive design system that includes design tokens, typography scales, color palettes, and a complete set of UI components. Every element is carefully crafted to work together seamlessly, ensuring consistency across your entire application.

Built on top of Angular Material 3, AegisX extends the foundation with additional components and patterns commonly needed in enterprise applications. Whether you're building dashboards, data tables, or complex forms, AegisX has you covered.

Key Features

Comprehensive Component Library

AegisX includes over 50+ components ranging from basic elements like buttons and inputs to complex components like data tables, date pickers, and charts. All components are built as standalone Angular components using the latest Angular features including Signals for reactive state management.

Design Token System

A complete design token system built with CSS custom properties provides:

  • Color tokens - Semantic colors with Zinc palette for better readability
  • Typography tokens - Two complete typography systems (UI & Prose)
  • Spacing tokens - Consistent spacing scale across all components
  • Shadow tokens - Elevation system for depth and hierarchy
  • Radius tokens - Border radius scale for rounded corners

Two Typography Systems

AegisX provides two specialized typography systems:

  1. AegisX Typography - Optimized for UI components with 16px base size and 1.5 line-height. Perfect for dashboards, forms, and data tables.
  2. Prose Typography - Optimized for long-form content with 15px base size and 1.7 line-height. Ideal for documentation, articles, and help pages.

"Good design is invisible. The best typography gets out of the way and lets the content shine." - AegisX Design Principles

Accessibility First

Every component in AegisX is built with accessibility in mind, following WCAG 2.1 AA guidelines:

  • Proper ARIA labels and roles
  • Keyboard navigation support
  • High contrast color combinations
  • Screen reader compatibility
  • Focus indicators and states

Getting Started

Ready to start building with AegisX? Here's how to get started:

1. Installation

First, install the required packages in your Angular project. Visit our Installation Guide for detailed step-by-step instructions. 

pnpm add @aegisx/ui @angular/material @angular/cdk

2. Configuration

Configure your Angular application to use AegisX theme and design tokens. The setup process takes just a few minutes and includes:

  • Importing the AegisX theme SCSS files
  • Configuring Material Design theme
  • Setting up design tokens
  • Importing required fonts (Inter, Noto Sans Thai)

3. Start Building

Once configured, you can start using AegisX components in your application. Check out our Quick Start Guide for examples and best practices.

Tip: Explore the Typography Guide to understand when to use each typography system in your documentation pages.

Component Categories

AegisX components are organized into logical categories:

Data Display

Components for displaying data including badges, cards, chips, KPI cards, and stat cards. Perfect for dashboards and analytics interfaces.

Forms & Input

Complete form controls including text inputs, selects, date pickers, checkboxes, radio buttons, and switches. All components support ReactiveFormsModule and ngModel.

Navigation

Navigation components including breadcrumbs, tabs, stepper, and sidebar navigation. Responsive and touch-friendly on all devices.

Feedback

User feedback components like alerts, dialogs, snackbars, and loading indicators. Keep users informed about system status and actions.

Data Tables

Advanced data table component with sorting, filtering, pagination, and row selection. Optimized for large datasets with virtual scrolling support.
