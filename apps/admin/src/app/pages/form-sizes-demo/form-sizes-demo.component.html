Usage Guidelines

Best Practices

  • check_circle Use .form-compact for Tremor-style CRUD interfaces
  • check_circle Use .form-standard as default for general forms
  • check_circle Maintain consistency within the same feature or page
  • check_circle Test on mobile devices (minimum 44px touch targets)

Common Mistakes

  • cancel Mixing different sizes in the same form
  • cancel Using .form-compact for primary user input forms
  • cancel Using .form-standard for dense data tables
  • cancel Ignoring accessibility requirements (minimum 44px touch targets on mobile)
