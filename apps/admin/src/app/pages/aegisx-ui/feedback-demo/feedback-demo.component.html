Loading Bar Component

Fully controllable loading bar with service-based architecture. Control from any component in your app!

Simple Loading (Indeterminate)

loadingBarService.show('primary')

Success Loading

loadingBarService.showSuccess('Success!')

Error Loading

loadingBarService.showError('Error!')

Warning Loading

loadingBarService.showWarning('Warning!')

Progress Loading (Simulated Upload)

loadingBarService.showProgress(0)
setProgress(percent)

Multi-Step Process

setProgress(33/66/100)
with step messages

Simulated API Call

async/await with
try/catch handling

Auto-Progress Simulation

simulateProgress(3000, 90)

The loading bar uses LoadingBarService for centralized control. Inject the service in any component to control the loading bar in the layout!

  • Service-based: Injectable via dependency injection
  • Reactive: Uses RxJS BehaviorSubject for state management
  • Type-safe: Full TypeScript support with interfaces
  • Flexible: Indeterminate & Determinate modes
  • Colors: primary, success, warning, error, neutral
