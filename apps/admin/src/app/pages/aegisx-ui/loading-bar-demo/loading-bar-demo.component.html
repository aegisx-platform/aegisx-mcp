API Reference

üìñ LoadingBarService Methods

Basic Methods
show(color?, message?) Show indeterminate loading bar
hide() Hide the loading bar
complete(delay?) Complete and hide after delay
Progress Methods
showProgress(percent, color?, message?) Show determinate progress
setProgress(percent, message?) Update progress percentage
simulateProgress(duration, maxPercent?) Auto-increment progress
Convenience Methods
showSuccess(message?) Show success state
showError(message?) Show error state
showWarning(message?) Show warning state

Best Practices

üí° Usage Guidelines

‚úÖ Do
  • ‚Ä¢ Use indeterminate loading for unknown duration operations
  • ‚Ä¢ Show progress percentage for file uploads/downloads
  • ‚Ä¢ Provide meaningful status messages during multi-step processes
  • ‚Ä¢ Hide loading bar when operation completes (success or error)
  • ‚Ä¢ Use appropriate colors (success for completion, error for failures)
  • ‚Ä¢ Call complete() for smooth transitions before hiding
‚ùå Don't
  • ‚Ä¢ Don't forget to hide() the loading bar after operations
  • ‚Ä¢ Don't use progress bar for instantaneous operations
  • ‚Ä¢ Don't show multiple loading bars simultaneously
  • ‚Ä¢ Don't use loading bar as a replacement for alerts/notifications
  • ‚Ä¢ Don't set progress > 100 or < 0
  • ‚Ä¢ Don't leave loading bar visible indefinitely
