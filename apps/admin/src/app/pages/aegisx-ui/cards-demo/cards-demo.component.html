API Reference

Property Type Default Description
variant 'default' | 'outlined' | 'elevated' 'default' Visual style: default (1px border), outlined (2px), elevated (shadow)
size 'sm' | 'md' | 'lg' 'md' Internal padding: sm (compact), md (standard), lg (spacious)
color 'default' | 'primary' | 'info' | 'success' | 'warning' | 'error' | 'neutral' 'default' Semantic color for borders and backgrounds
colorIntensity 'filled' | 'subtle' 'filled' Color intensity: filled (full color) or subtle (light tint)
borderWidth '1' | '2' | '3' | '4' | string '1' Border thickness (predefined or custom like "2.5px")
hoverable boolean false Enables lift animation on hover
clickable boolean false Adds pointer cursor, press effect, keyboard accessibility
loading boolean false Shows loading spinner overlay
title string '' Card header title text
subtitle string '' Card header subtitle text
